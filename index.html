Create a Procedural Mesh

Create a mesh at runtime using the Procedural Mesh Component. This example exposes 3 vectors to the editor that allow you to change the vertices in the editor.
Tip: You can expose variables as "3D widgets" that can be maniuplated at runtime by enablind the "Show 3D widget" option in the blueprint.
